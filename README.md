## Introduction

This is a repository intended for DSA studies, the codes are based on two books:

- Algorithms: Theory and Practice by Thomas H. Cormen
- Data Structure & Algorithms in Java by Michael T. Goodrich, Roberto Tamassia and Michael H. Goldwasser

The technologies that are in use are located below the main title and will be frequently updated as they are used.

## Motivation

I consider DSA to be the most important area of Software Engineering and Computer Science, from implementing a Linked List to solving NP-Complete problems using Dynamic Programming, but if you want some reasons to study, here are a few:

Performance Efficiency, Efficient Problem Solving, Resource Optimization, Solving Real World Problems, Understanding Frameworks and Libraries, etc.
